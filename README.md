# Food Tracker

Simple offline-first food log with a daily whole-foods target.

## Local preview

```sh
cd /Users/evanwallis/Coding Projects/food-tracker
python3 -m http.server 8080
```

Open http://localhost:8080.

## Deploy

```sh
vercel --prod
```
